Jay Chou Data Analysis 周杰倫數據分析 (2000-2016)

by Brian Cheng, Sep 2020

This is a Python project where I used Spotify API data and data I compiled myself to analyze songs from Jay Chou's studio albums (up to 2016).

Code where I obtained, cleaned, and analyzed data in more detail 為數據採取，整理，與更深度分析所寫的程式: (github.com/brc042/Data-Analysis-on-Jay-Chou)

Here are some of my key takeaways 以下是我分析後的幾個重點:



1. Jay Chou's most-frequent collaborating lyricist is 方文山 and most-frequent collaborating arranger is 林邁可.

周杰倫最常合作的作詞人是方文山，最常合作的編曲人是林邁可。





2. The longer a Jay Chou song is, the more likely that it will be popular on Spotify.

越長的周杰倫歌曲越有可能在Spotify上有高的人氣。



3. Jay Chou's most used keys are C and D major. Jay Chou hasn't used F#M/GbM in his studio albums yet as of 2016.

周杰倫最常用的調性是C和D大調。周杰倫至今 (2016) 還未在主專輯裡使用過F#M/GbM大調。
